# SQLSprint4

Sprint4 Job Application Database

This project demonstrates the creation and use of a relational SQL database designed to manage job applications. The database includes tables for Users, Companies, Jobs, and Applications, allowing users to apply for jobs and store information about companies and positions. The project demonstrates common database operations including creating tables, inserting data, querying information, updating records, deleting records, and performing joins between related tables.

**Instructions for Build and Use**
Steps to build and/or run the software

Open MySQL Workbench.

Run the provided SQL script to create the Sprint4 schema and tables.

Execute the remaining SQL statements to populate the tables with sample data.

Instructions for using the software

Use SELECT queries to view job listings, companies, and user applications.

Use INSERT statements to add new users, companies, jobs, or applications.

Use UPDATE and DELETE statements to modify or remove records from the database.
